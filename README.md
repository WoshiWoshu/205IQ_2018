# 205IQ_2018
2nd year mathematics computer science mini project written in C++.

PREREQUISITE

    gnuplot need to be installed

USAGE

    ./205IQ u s [IQ1] [IQ2]
    
DESCRIPTION

    u       mean
    s       standard deviation
    IQ1     minimum IQ
    IQ2     maximum IQ
    
EXEMPLE

    ./205IQ 100 24 90
    
    > 33.8% of people have an IQ inferior to 90
    
    ./205IQ 100 24 90 95
    
    > 7.9% of people have an IQ between 90 and 95
